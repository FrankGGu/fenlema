# 关系晴雨表 - 情侣关系管理小程序

## 项目概述

关系晴雨表是一款专为情侣设计的关系管理小程序，通过每日打卡、情绪记录、AI调解、修复任务等功能，帮助情侣更好地维护和改善关系。

## 核心功能

### 1. 每日关系打卡
- 心情评分（1-5星）
- 沟通状态评估
- 亲密度记录
- 冲突情况记录
- 备注说明

### 2. 关系温度计
- 可视化关系健康度（0-100分）
- 温度变化趋势图
- 连续打卡天数统计

### 3. 智能预警系统
- 漏打卡提醒
- 低评分预警
- 冲突升级预警
- 双方冷淡检测

### 4. AI调解室
- 智能对话分析问题
- 矛盾类型识别（冷战、误解、信任问题等）
- 个性化修复建议
- 生成修复任务

### 5. 修复任务系统
- AI生成的个性化任务
- 任务进度跟踪
- 完成任务获得关系温度奖励
- 任务分类（倾听、感谢、约定、回忆等）

### 6. 心情树洞
- 匿名情绪记录
- 情绪标签分类
- 历史情绪回顾
- 情绪统计分析

### 7. 疗愈内容
- 放松音频（呼吸练习、冥想等）
- 关系修复文章
- 正念练习指导
- 关系建议卡片

### 8. 关系日历
- 每日打卡记录可视化
- 心情变化趋势
- 冲突频率统计
- 月度关系报告

## 技术架构

### 前端技术栈
- 微信小程序原生框架
- WXML + WXSS + JavaScript
- 组件化开发
- 本地数据存储

### 项目结构
```
/
├── app.js              # 小程序入口文件
├── app.json            # 全局配置
├── app.wxss            # 全局样式
├── components/         # 公共组件
│   ├── checkin-card/           # 打卡卡片组件
│   ├── relationship-thermometer/ # 关系温度计组件
│   ├── ai-message/             # AI消息组件
│   └── task-card/              # 任务卡片组件
├── pages/              # 页面文件
│   ├── index/                  # 首页
│   ├── checkin/                # 打卡页面
│   ├── calendar/               # 日历页面
│   ├── warning/                # 预警页面
│   ├── ai-mediation/           # AI调解室
│   ├── task/                   # 任务页面
│   ├── mood/                   # 心情树洞
│   ├── healing/                # 疗愈内容
│   ├── profile/                # 个人中心
│   └── login/                  # 登录页面
└── utils/              # 工具函数
    ├── util.js                 # 通用工具函数
    ├── constant.js             # 常量定义
    ├── mock-data.js            # 模拟数据
    └── request.js              # 网络请求封装
```

### 数据管理
- **全局状态管理**: 使用小程序全局变量
- **本地存储**: wx.setStorageSync / wx.getStorageSync
- **模拟数据**: 完整的模拟数据系统，支持离线使用
- **数据持久化**: 自动保存用户数据到本地

## 核心业务逻辑

### 1. 打卡系统
- 每日一次打卡
- 四项维度评分
- 自动计算关系温度变化
- 连续打卡奖励机制

### 2. 预警算法
- 基于打卡数据的实时分析
- 多维度风险评估
- 分级预警机制
- 智能处理建议

### 3. AI调解流程
- 自然语言对话分析
- 矛盾类型识别
- 个性化建议生成
- 修复任务创建

### 4. 任务系统
- 任务状态管理（待完成/进行中/已完成）
- 任务过期提醒
- 完成任务奖励
- 任务统计报告

## 使用流程

1. **用户登录** → 2. **首页查看关系状态** → 3. **每日打卡** → 4. **查看预警提醒** → 5. **使用AI调解** → 6. **完成修复任务** → 7. **记录心情** → 8. **查看疗愈内容** → 9. **查看关系日历**

2. 配置 AppID（测试时可使用测试号）

## 设计理念

关系晴雨表的设计理念是"预防胜于治疗"，通过日常的微小行动和及时的干预，帮助情侣建立更健康、更持久的亲密关系。我们相信，良好的关系需要持续的关注和努力，而科技可以成为这个过程中的有力助手。
